# EmailClient-Nodejs
A tryout email client developed based on the latest GMAIL API .Nothing new
\n
How to use?
1.Install node js \n
2.Clone this repository to your local pc . \n
3.Keep port no 1234 open \n
4.Open terminal in the directory and type "node em" \n
5.Open "localhost:1234" in your browser \n
6.Enjoy \n
